## Notes

- בשלב הזה ייתכנו עדיין עותקים ישנים של חלק מהמסמכים בשורש הריפו.
- הגרסאות שבתוך `Docs/` אמורות להפוך בהדרגה למיקום הראשי.
- מומלץ ליצור מסמכים חדשים ישירות תחת אחת הקטגוריות המתאימות.

## Rule of thumb

- Workflow → תהליך עבודה, פלאג-אין, GitHub, ChatGPT
- Testing → תוכניות בדיקה, QA, test plans
- Design → באלאנס, עקרונות גיימפליי, תבניות אויבים
- Planning → roadmap, next steps, milestone notes
- Tech → הערות ארכיטקטורה, מבנה קוד, ownership
